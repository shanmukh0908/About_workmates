Data Handling

Concurrent Direct-to-Cloud Uploads

Built a client-side image upload flow using Promise.all() and FormData to avoid backend load.

more... ✖
Engineering Strategy:
Instead of sending large images to the backend, the frontend first requests a signature.

After authorization, multiple FormData requests are sent in parallel directly to Cloudinary.

Impact:
• Faster uploads
• No UI blocking
• Better handling of async operations
React • Axios • Promise.all • FormData • Cloudinary API

📡 Centralized API Layer with Axios

Network Architecture

Centralized API Layer

Created a reusable API layer using a custom Axios instance to handle authentication and error handling in one place.

more... ✖
Engineering Strategy:
Used request interceptors to attach JWT tokens and response interceptors to handle common errors like 401.

This avoids repeating auth logic in multiple components.

Impact:
• Cleaner code
• Centralized error handling
• Easier to maintain API logic

Redux State Management

State Separation Strategy

Used Redux Toolkit for global state and local state for component-level logic.

more... ✖
Engineering Strategy:
Redux was used only for data that needs to persist across pages like user info and roles.

Impact:
• Predictable global state
• Reduced unnecessary complexity
• Better separation of concerns
